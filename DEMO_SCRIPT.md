# OMFS Case Coach — 1-Minute Demo Script

**Audience**: Professor / Faculty  
**Purpose**: Demonstrate the three core study modes  
**Duration**: ~60 seconds

---

## Setup (before demo)
- App is open at omfs-case-coach.vercel.app
- Tab is on Case Coach

---

## Scene 1 — Case Coach (0:00–0:25)

> "Let me show you what a student sees. They come to Case Coach and paste in a clinical case."

**Click "Load sample case"** → the 27-year-old case loads into the text box.

> "This is a patient with right lower jaw swelling — pericoronitis with possible Ludwig's angina. The student hits Analyze Case."

**Click Analyze Case** → wait for response.

> "The AI gives a structured breakdown: the case summary, key clinical findings, and critically — what information is MISSING. Then it flags the red flags, like the floor of mouth swelling and restricted opening, and lists what an instructor might actually ask them on rounds."

---

## Scene 2 — Socratic Mode (0:25–0:45)

> "Now in Socratic Mode, instead of giving the answer, it guides them."

**Click Socratic Mode tab** → select the Ludwig's angina starter case.

**Click Start Socratic Session**.

> "It asks one question at a time — 'What is your main concern with this patient?' The student answers, gets feedback, and the next question deepens the reasoning. It's like a virtual attending walking rounds with them."

---

## Scene 3 — Exam Mode (0:45–1:00)

> "And for board prep, Exam Mode pulls questions from your actual lecture notes."

**Click Exam Mode tab** → click "Odontogenic infections & fascial spaces".

> "It generates a case-based question — MCQ or short answer. The student answers, and it explains exactly why each option is right or wrong, and cites the lecture file it drew from."

**Point to the source citation in the response.**

> "All three modes run off the lecture files you place in the repository — no uploading required, no student accounts, no complex setup. They add their notes to a folder and it's live."

---

## Key Points to Emphasize
- **Source-grounded**: Cites your lecture files by name
- **Safety framed**: Every response reminds students this is for study only
- **Zero student setup**: Just open the URL
- **Extensible**: Add any lecture as a `.md` file — no code changes needed
